# CS50 Web Programming Projects

This repository contains my assignments and final project completed as part of **CS50’s Web Programming with Python and JavaScript**.  
Each folder represents a course project covering different aspects of modern web development — from front-end design to back-end integration and database management.

---

## How to Run

1. **Clone the repository**

   ```bash
   git clone https://github.com/<your-username>/cs50_web_prashidika.git

   ```

2. **Navigate to any project folder**

   ```bash
   cd commerce

   ```

3. **Run migrations and start the Django server**

   ```bash
   python manage.py migrate
   python manage.py runserver

   ```

4. **Open your browser**
